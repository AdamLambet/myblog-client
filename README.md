# 介绍：\
## 本系统主要分为两大模块： \
**（1）博客系统主体**\
    主要技术栈：react+mongodb+mysql\
    主要功能：\
            * 实现个人信息展示\
            * 实现博客的权限，登陆/注册/修改密码\
            * 实现博客文章列表的展示\
            * 实现文章添加/编辑/删除\
**（2）富文本编辑器**\
    基于typescript实现一个通用的富文本编辑器\
    主要功能：\
            * 实现基础的文字输入功能\
            * 实现toolbar，有文字加粗，字体大小，字体颜色，undo/redo\
            * 能支持当前富文本编辑器内容导出为htmlString/json形式\
